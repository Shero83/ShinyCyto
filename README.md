## cyjShiny

cytoscape.js as a shiny widget, with an API based on RCyjs (and ancestrally, RCy3)

## Installation (Quick Version)

```
library(remotes)
remotes::install_github(repo="cytoscape/cyjShiny", ref="master", build_vignette=TRUE)
```
